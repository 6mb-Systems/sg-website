# SEO for Marketing Site & Blog

Guidance for implementing SEO correctly across pages and blog posts.

## When to Use

- Creating new pages
- Implementing blog listing or post pages
- Adding metadata, sitemap, or robots rules

## Instructions

- Every page must define:
  - title
  - description
- Blog posts should also include:
  - OpenGraph metadata
  - Published date
  - Canonical URL
- Use semantic headings (one H1 per page).
- Add structured data (Article) to blog posts when possible.
- Ensure sitemap includes:
  - Static pages
  - Blog index
  - Individual blog posts
- Use alt text for meaningful images.
- Avoid heavy client-side JavaScript on marketing pages.
- Ask clarifying questions if SEO priorities or target audiences are unclear.
